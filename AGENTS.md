# QuizMe
QuizMe - Create and take quizzes.

## Project Overview
QuizMe is a web application that allows users to create and take quizzes. It is built using React and TypeScript. It is designed to be responsive, user-friendly, intuitive and playful. We are following the Neo-brutalism design principles.

### Features
- Create quiz based on the user input topic and difficulty level.
- Take quiz and get score.
- Get explanation for each incorrect answer.

### Technologies Used
- React (https://react.dev/reference/react)
- TypeScript (https://www.typescriptlang.org/docs/)
- Tailwind CSS (https://tailwindcss.com/docs/installation/using-vite)
- shadcn/ui (https://ui.shadcn.com/docs/installation)
- Neo-brutalism (https://www.neobrutalism.dev/docs)

Always refer to the documentation in the respective links if required.
For UI components, only import the component if required.

## Implementation Details

### Build
The project use bun JavaScript runtime. Always use bun to build the project.
Refer to the documentation at https://bun.com/docs if required for more information.

Commands
- `bun run dev`: Start the development server.
- `bun run build`: Build the project for production.
- `bun run start`: Start the production server.
- `bun run test`: Run tests.
- `bun run lint`: Run linting.
